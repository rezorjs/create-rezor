# create-rezor

创建 Rezor 小程序项目的最佳方案。

## 使用

要使用 `create-rezor` 创建一个全新的 Rezor 小程序项目，请直接在终端内运行以下命令：

```bash
npm create rezor@latest
```

<!-- prettier-ignore -->
> [!NOTE]
> `@latest` **不能**省略，否则 `npm` 可能会使用已缓存的旧版本。
